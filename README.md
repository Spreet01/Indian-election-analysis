Indian Election Data Analysis

Project Overview
This project analyzes Indian election data to uncover insights into voting patterns, party performance, and constituency-level trends.
The goal is to visualize and understand how elections shape political outcomes across regions.

Features

1. State-wise election results analysis
2. Party-wise performance comparison
3.  Vote share vs seat share analysis
4.  Trend analysis across years (if dataset includes multiple elections)
5.  Constituency-level insights
6.  

Tools & Technologies
* Power BI 
* Excel / CSV dataset
* Data Cleaning & Transformation
* Data Visualization

 Key Insights
* Identified leading political parties by seats and vote share
* Highlighted regional strongholds of major parties
* Compared vote share vs actual seat conversion
* Analyzed close contests and winning margins
